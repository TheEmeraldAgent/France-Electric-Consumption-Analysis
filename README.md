# France-Electric-Consumption-Analysis

Data analysis project exploring electricity consumption patterns in France using Python with exploratory data analysis and data visualization.

## Project Overview

This project analyzes electricity consumption data in France to understand usage patterns and trends over time.  
The analysis focuses on exploring consumption distribution, identifying patterns, and visualizing electricity usage across different periods.

The notebook includes data cleaning, exploratory data analysis (EDA), and visualization to gain insights into electricity demand behavior.

## Dataset

The dataset contains electricity consumption data in France including variables such as:

- date
- electricity consumption
- regional or national usage levels
- time-based consumption patterns

These variables are used to analyze consumption trends and electricity demand behavior.

## Technologies Used

Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Jupyter Notebook

## Analysis

The notebook performs:

- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Visualization of electricity consumption patterns  
- Trend analysis of electricity demand
